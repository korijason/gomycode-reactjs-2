# FIFA Player Cards

This is a simple React application that displays FIFA player cards. Each player card contains the player's name, team, nationality, jersey number, age, and an image.

## Features
- Displays player cards using react-bootstrap.
- Data is passed through props to a Player component.
- Default props are defined in the Player component.
- Player data is stored in an external JSON array (`players.js`).

## Requirements
- Node.js
- React
